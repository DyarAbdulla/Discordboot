# Step-by-Step: Find GitHub URL in Railway

## üéØ You're on the Settings Page - Here's Where to Look:

### Step 1: Scroll Up on Settings Page
Look at the **top** of the Settings page for:
- "Source" section
- "Repository" field
- GitHub icon with URL

### Step 2: Check Left Sidebar
On the **left sidebar**, you see "Discordboot" card:
- Click the **GitHub icon** (Octocat) next to "Discordboot"
- This might show the repository info

### Step 3: Check Other Tabs
Try clicking these tabs:
- **"Deployments"** tab ‚Üí Click a deployment ‚Üí Look for "Source"
- **"Variables"** tab ‚Üí Sometimes shows repo info
- **Back to main view** ‚Üí Click service name ‚Üí Shows connection

### Step 4: Alternative - Check Project Level
1. Click **"cooperative-contentment"** at the top (project name)
2. Go to project **Settings**
3. Look for **"Source"** or **"Repository"**

## üîç What You're Looking For:

The URL will look like:
```
https://github.com/your-username/discord-bot
```

OR

```
git@github.com:your-username/discord-bot.git
```
